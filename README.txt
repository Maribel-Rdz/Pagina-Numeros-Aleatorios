Página Números Aleatorios
---------------------------------------
Este trabajo consiste en mostrar números aleatorios que cumplan con los siguientes
criterios:

1. Números enteros (4 digitos)
2. Números decimales (4 dígitos enteros, 3 dígitos decimales)
3. Números hexadecimales (4 dígitos)
4. Números binarios (8 dígitos)
